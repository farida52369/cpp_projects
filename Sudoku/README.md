## [Sudoku game](https://en.wikipedia.org/wiki/Sudoku)

- Application developed using:
  - C++ v11
  - Qt5 

<br>

- How To use:
  - Three levels are provided through ComboBox to Start New Game: 
      - Easy
      - Meduim
      - Hard
  
  - Undo
  - Solve The Game (The solution depends on Backtracking Algorithm)
